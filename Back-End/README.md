# Backend - Projeto Mais Saúde

API desenvolvida com Laravel responsável pelas regras de negócio e manipulação de dados.

---

## 📌 Funcionalidades

- Cadastro e autenticação de usuários
- Validação de dados nas requisições
- Estrutura em controllers e models
- Rotas organizadas via API

---

## 🧠 Tecnologias

- PHP
- Laravel
- Eloquent ORM
- Banco de dados relacional

---

## ▶️ Como executar

composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
